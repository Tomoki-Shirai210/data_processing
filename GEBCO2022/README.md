# GEBCO decode script
- GEBCO Netcdf data(2022) -> ascii form (Longitude, Latitude, elevation) conversion

## Requirements
- Netcdf4 (for python)

## Data 
You can download the GEBCO global elevation data: 
https://www.gebco.net/data_and_products/gridded_bathymetry_data/
